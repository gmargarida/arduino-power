arduino-power
=============

Arduino project to control the powerswitch tail.

Connection instructions:
========================

* Connect ground from powerswitch tail to arduino.
* Connect -in from powerswitch tail to ground on arduino.
* Connect +in from powerswitch tail to pin 3 on arduino.

Open arduino IDE and upload the project to your arduino.
Use the serial monitor to interface with the powerswitch tail.
The following commands are available:
* enable
* disable
* status
* reset (resets for 2 seconds)
